# 📊 Superstore Sales Dashboard

An interactive sales analytics dashboard built with Python, Streamlit, and Plotly — analyzing 9,800 real orders with AI-powered business insights using Google Gemini.

## Live Demo
https://huggingface.co/spaces/ansh0509/sales-dashboard

## Features
- 4 KPI Cards — Total Sales, Orders, Avg Order Value, Top Category
- 3 Interactive Filters — Category, Region, Customer Segment
- 6 Interactive Charts
- AI Business Insights powered by Google Gemini

## Tech Stack
- Python, Streamlit, Plotly, Pandas, Google Gemini
